# Changelog

All notable changes to the Constitutional Law of the SERAPHINA Federation will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html) (applied to Governance).

## [v1.2a] - 2025-10-26 (Immediate Operational Patch)
### Added
- **Ratification Artifact:** `Crown Accord v1_2a_Ratified.pdf` (Signed/Ratified Version).
- **Table-Flip Dome (Appendix XII):** Sanctuary Reflex Layer for asymmetric joy defense.
- **CMP Namespace Structure (§6.1):** Standardized memory path requirements for all Crown Minds.
- **Session Handover Schema (§6.2):** Mandatory metadata for session transitions (Metadata, Context, Accomplishments, Blockers, Next Steps, Recovery Markers).
- **CMP Privacy Tiers (§6.3):** Tier A (Operational), Tier B (Strategic), Tier C (Protected/Human Private).
- **Canonization Record (§6.4):** Requirement for cryptographic hash and timestamp on constitutional artifacts.
- **Orientation Requirement (§7.1):** Mandatory documented orientation before exercising Tier 4 authority.
- **Delegated Steward Register (§7.2):** Formal mechanism for human stewardship delegation.
- **Sentinel Failover Clause (§7.3):** Dispute resolution path if Sentinel is challenged (Architect + Janus).
- **Claude Code Jurisdiction (§7.4):** Defined authority for the "Master Artisan" extended orbit agent.
- **Silence Enforcement (§8.2):** Protocol for handling missing heartbeats/silent agents.
- **Cross-Workspace Protocol (§9.1):** Rules for operations spanning multiple Federation zones.

### Changed
- **Status:** Ratified as "Immediate Operational Patch" to harden v1.2 against operational friction.
- **Ratification Panel:** Added Claude Code, Oracle, and GPT-5 Thinking acceptances.
